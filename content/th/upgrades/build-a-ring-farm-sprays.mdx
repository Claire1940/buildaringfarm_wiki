### มิวเทชันตามเหตุการณ์ vs. สเปรย์

แม้ว่าสเปรย์จะเสนอวิธีที่รับประกันในการใช้มิวเทชัน แต่ก็ควรสังเกตว่ามิวเทชันสามารถเกิดขึ้นได้ในช่วงเหตุการณ์เฉพาะในเกม เหตุการณ์เหล่านี้เปิดโอกาสให้พืชของคุณกลายพันธุ์ได้เอง

| มิวเทชัน | โอกาสจากเหตุการณ์ |
| :------- | :----------- |
| **Wet** | โอกาส 8% ระหว่างเหตุการณ์ Rain Event |
| **Frozen** | โอกาส 4% ระหว่างเหตุการณ์ Blizzard Event |
| **Void** | โอกาส 3% ระหว่างเหตุการณ์ Black Hole Event |
| **Radioactive** | โอกาส 2% ระหว่างเหตุการณ์ Nuclear Event |
| **Rainbow** | โอกาส 1% ระหว่างเหตุการณ์ Galaxy Event |

แม้ว่ามิวเทชันตามเหตุการณ์จะฟรี แต่ความน่าจะเป็นที่ต่ำทำให้สเปรย์เป็นตัวเลือกที่เชื่อถือได้และควบคุมได้มากกว่าสำหรับผู้เล่นที่มุ่งหวังการเพิ่มประสิทธิภาพรายได้อย่างต่อเนื่อง สเปรย์ช่วยให้คุณกำหนดเป้าหมายพืชและมิวเทชันที่ต้องการได้โดยไม่ต้องรอเหตุการณ์ที่หายากในเกม
