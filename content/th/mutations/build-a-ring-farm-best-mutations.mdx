### การทำงานของการกลายพันธุ์

มีการกลายพันธุ์หลายประเภท ซึ่งแต่ละประเภทมีตัวคูณและลักษณะที่แตกต่างกัน สามารถหาได้จากกิจกรรมในเกมหรือโดยการใช้สเปรย์เฉพาะที่ซื้อจาก Gear Shop การทำความเข้าใจผลกระทบและวิธีการได้มาของการกลายพันธุ์แต่ละประเภทเป็นสิ่งสำคัญสำหรับการเพิ่มผลผลิตของฟาร์มของคุณให้สูงสุด

นี่คือรายละเอียดของการกลายพันธุ์ที่ดีที่สุด ตัวคูณ และวิธีการได้มา:

| **แช่แข็ง (Frozen)**  | รายได้ **1.75x** ลักษณะแช่แข็ง                     | โอกาส 4% ในช่วงกิจกรรมพายุหิมะ ใช้สเปรย์แช่แข็ง [$750M]                                             |
| **ว่างเปล่า (Void)**    | รายได้ **2.25x** ลักษณะสีดำ                      | โอกาส 3% ในช่วงกิจกรรมหลุมดำ ใช้สเปรย์ว่างเปล่า [$1B]                                               |

| **สายรุ้ง (Rainbow)** | รายได้ **5x** สีที่เปลี่ยนแปลงได้                  | โอกาส 1% ในช่วงกิจกรรมกาแล็กซี ใช้สเปรย์สายรุ้ง [$1T]                                                |

### การอัปเกรดที่สำคัญสำหรับฟาร์มไม่จำกัด

| หมวดหมู่การอัปเกรด | ลำดับความสำคัญ | ผลกระทบต่อประสิทธิภาพการกลายพันธุ์                                                                |
| :--------------- | :------- | :------------------------------------------------------------------------------------------- |
| **พลังสปริงเกลอร์** | สูง     | ควบคุมความเร็วในการเติบโตของพืชโดยตรง ทำให้มั่นใจได้ว่าพืชกลายพันธุ์จะถูกเก็บเกี่ยวอย่างต่อเนื่อง          |
| **ระยะสปริงเกลอร์** | สูง     | เพิ่มความครอบคลุม ทำให้พืชกลายพันธุ์จำนวนมากขึ้นได้รับประโยชน์จากการเติบโตที่เร่งขึ้น          |
| **ระยะเลื่อย**    | ปานกลาง   | เก็บเกี่ยวอัตโนมัติอย่างมีประสิทธิภาพมากขึ้น ป้องกันความล่าช้าสำหรับพืชกลายพันธุ์ที่มีมูลค่าสูง       |
| **โชคเมล็ดพันธุ์**    | สูง     | เพิ่มโอกาสในการได้พืชที่แข็งแกร่งขึ้น ซึ่งเป็นตัวเลือกที่เหมาะสำหรับการกลายพันธุ์       |
| **พื้นที่ฟาร์ม**   | ต่ำ (ช่วงต้น) | ขยายเมื่อการตั้งค่าปัจจุบันได้รับการปรับปรุงแล้วเท่านั้น การขยายก่อนกำหนดจะทำให้ความก้าวหน้าโดยรวมช้าลง |

### พืชระดับสูงสำหรับการกลายพันธุ์

เมื่อโชคของคุณดีขึ้น คุณจะปลดล็อกเมล็ดพันธุ์ที่มีค่ามากขึ้น นี่คือตัวเลือกหลักสำหรับการกลายพันธุ์ที่คุณหามาอย่างยากลำบาก:

| ชนิดพืช          | ศักยภาพมูลค่า (กลายพันธุ์) | หมายเหตุ                                                                                   |
| :----------------- | :------------------------ | :-------------------------------------------------------------------------------------- |
| **เมล่อน/แตงโม** | สูง                      | ตัวเลือกที่ดีในช่วงต้น-กลางเกมสำหรับการกลายพันธุ์                                               |
| **พีช/เห็ด** | สูง                      | ตัวเลือกที่แข็งแกร่งก่อนที่จะปลดล็อกเมล็ดพันธุ์ที่หายากกว่า                                             |
| **เห็ดเรืองแสง (Glow Shroom)**    | สูงมาก                 | ยอดเยี่ยมสำหรับการกลายพันธุ์ โดยเฉพาะอย่างยิ่งกับ Void หรือ Radioactive                            |
| **แอปเปิ้ลทองคำ (Golden Apple)**   | สูงมาก                 | พืชระดับสูงสุด แอปเปิ้ลทองคำกลายพันธุ์สามารถสร้างรายได้หลายพันล้าน                           |
| **กีวี (Kiwi)**           | สูงมาก                 | ตัวเลือกหลักอีกตัวสำหรับการกลายพันธุ์ Rainbow ให้ผลกำไรมหาศาล                 |
| **ซากุระ (Cherry Blossom)** | สูงมาก                 | มีค่ามาก โดยเฉพาะอย่างยิ่งเมื่อรวมกับการกลายพันธุ์ที่ทรงพลัง                        |
| **ลับ/ศักดิ์สิทธิ์/แปลกใหม่ (Secret/Divine/Exotic)** | ศักยภาพล้านล้าน       | เน้นช่วงท้ายเกม พืชเหล่านี้จะเพิ่มขนาดได้อย่างเหลือเชื่อด้วยการกลายพันธุ์และการอัปเกรด         |

<Callout type="tip" title="การข้ามเวลาเพื่อการจ่ายเงินจำนวนมาก">
ยิ่งพืชของคุณแข็งแกร่งก่อนที่จะเปิดใช้งานการข้ามเวลา การจ่ายเงินก็จะยิ่งมากขึ้นทันที ผู้เล่นที่มีพืชกลายพันธุ์ ลับ หรือแปลกใหม่สามารถสร้างรายได้หลายพันล้านหรือแม้กระทั่งล้านล้านจากการข้ามเวลาครั้งเดียวเมื่อฟาร์มของพวกเขาได้รับการอัปเกรดเต็มที่
