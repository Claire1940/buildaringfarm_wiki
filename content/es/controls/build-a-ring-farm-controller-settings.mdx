Aquí tienes un desglose de las configuraciones más impactantes y cómo configurarlas:

| Categoría de Configuración | Acción Recomendada | Beneficio |
| :--------------- | :----------------- | :------ |
| **Control de Volumen** | Ajustar a un nivel cómodo | Evita distracciones de audio, asegura que se escuchen sonidos importantes |
| **Modo Gráfico** | Manual y Bajo (si hay lag) | Reduce el lag, mejora la velocidad de fotogramas para un juego más fluido |
| **Vibración** | Apagado | Elimina la molesta vibración del control, mejora el enfoque |
| **Reducir Movimiento** | Activado (si hay lag) | Reduce aún más el lag visual, especialmente en consolas antiguas |
| **Invertir Cámara** | Apagado (estándar) | Mantiene controles de cámara intuitivos |
| **Sensibilidad** | Rango medio (preferencia personal) | Equilibra giros rápidos con puntería precisa |

<Section icon={Trophy} title="Guía Paso a Paso para Ajustar la Configuración">
Sigue estos sencillos pasos para configurar rápidamente la configuración de tu control en Build A Ring Farm para un rendimiento óptimo en tu PS4 o PS5.

<Steps>
  <Step num="1" title="Inicia Build A Ring Farm">
    Inicia la experiencia Build A Ring Farm en Roblox desde tu consola PS4 o PS5. Asegúrate de que el juego esté completamente cargado para acceder a todas las funciones.
  </Step>
  <Step num="2" title="Abrir Menú de Configuración">
    Una vez en el juego, presiona el **panel táctil** de tu control de PlayStation. Esto abrirá un menú rápido. Desde allí, presiona el botón **Opciones** para abrir la interfaz principal de configuración.
  </Step>
  <Step num="3" title="Navegar a Configuración de Pantalla y Control">
    En el menú de configuración, busca el icono de 'Configuración', generalmente ubicado en la esquina superior izquierda. Haz clic en él para revelar una lista de opciones ajustables, incluyendo gráficos, sonido y preferencias de control.
  </Step>
  <Step num="4" title="Configurar Gráficos y Rendimiento">
    - **Modo Gráfico**: Cambia esto a 'Manual' y desliza la barra de calidad a 'Bajo' si estás experimentando caídas de fotogramas o lag.
    - **Reducir Movimiento**: Activa esta opción para suavizar aún más el rendimiento visual, especialmente en sistemas más antiguos.
  </Step>
  <Step num="5" title="Ajustar Configuración de Control y Comodidad">
    - **Vibración**: Localiza la opción 'Vibración' y desactívala para una experiencia de juego más enfocada.
    - **Sensibilidad**: Ajusta el deslizador de 'Sensibilidad' a tu nivel preferido. Una configuración alrededor del medio es generalmente un buen punto de partida para un control equilibrado en Build A Ring Farm.
    - **Invertir Cámara**: Asegúrate de que 'Invertir Cámara' esté 'Apagado' a menos que prefieras específicamente los controles de cámara invertidos.
  </Step>
</Steps>
