Accessing your settings is simple: press the **Options button**, then navigate to the top-left corner to find the "Settings" menu.

<Callout type="warning" title="Lagging? Adjust Graphics!">
If you're experiencing significant lag, switch your graphics mode to **Manual** and set the graphics quality to **Low**. This can drastically improve performance and make your farming experience much smoother.
</Callout>

Consider these key settings for optimal play:

| Setting Category | Recommended Adjustment | Benefit |
| :--------------- | :--------------------- | :------ |
| **Graphics Mode** | Manual & Low | Reduces visual strain, improves frame rate, especially on older consoles or busy servers. |
| **Volume** | Adjust to Preference | Ensures game sounds are at a comfortable level. |
| **Vibration** | Off | Eliminates controller rumbling, which some players find distracting or annoying during long sessions. |
| **Reduce Motion** | On | Helps mitigate motion sickness and can slightly improve performance by simplifying visual effects. |
| **Camera Invert** | Off (Default) | Keeps camera controls standard; turning it on reverses vertical camera movement. |
| **Sensitivity** | Middle Range | Provides a balanced camera speed, allowing for precise movements without being overly fast or slow. |

<Section icon={Shield} title="Troubleshooting Common Control Issues">
Even with optimal settings, you might encounter minor control hiccups. This section addresses common problems players face with PS4/PS5 controls in Build A Ring Farm and provides quick solutions to get you back to farming.

| Issue | Potential Cause | Solution |
| :---- | :-------------- | :------- |
| **Unresponsive Buttons** | Controller connection issues, low battery | Reconnect controller, charge battery, or try a different controller. |
| **Cursor Jumps/Erratic** | High sensitivity, controller drift | Lower sensitivity in settings, or recalibrate/replace controller if drift persists. |
| **Laggy Interactions** | High graphics settings, poor internet | Reduce graphics to Manual & Low, check internet connection stability. |
| **Camera Inverted** | Invert Camera setting enabled | Disable "Camera Invert" in the game settings menu. |
| **Unable to Sell/Plant** | Insufficient funds, incorrect item selected | Ensure you have enough money, or verify the correct item is highlighted with R1/cursor. |

<Callout type="info">
If you're still experiencing persistent issues, consider restarting the Roblox application or even your console. Sometimes, a fresh start can resolve minor software glitches affecting game controls.
</Callout>

Remember that game updates can sometimes alter control mappings or introduce new options. Always check the in-game tutorial or official game announcements for the latest information.
