  </Step>
  <Step num="2" title="Use Sprays de Mutação">

</Steps>

<Section icon={Map} title="Gerenciando Mutações e Estratégias Avançadas">
Embora as mutações sejam poderosas, há nuances em seu gerenciamento. As plantas só podem ter uma mutação por vez, e às vezes você pode querer remover uma mutação para tentar uma melhor.

| Ação         | Item/Método              | Custo/Chance                               |
| :------------- | :----------------------- | :---------------------------------------- |
| **Aplicar Mutação** | Spray Específico         | Varia por spray (ex: $10M para Spray Úmido) |
|                | Evento no Jogo           | Varia por evento (ex: 8% para Evento Úmido) |
| **Remover Mutação** | Spray Ácido (Loja de Equipamentos) | **$1M**                                   |

Esta tabela destaca as ferramentas disponíveis para o gerenciamento de mutações. O Spray Ácido é uma maneira relativamente barata de reiniciar uma planta, permitindo que você tente uma mutação mais desejável, especialmente se você conseguir uma de nível baixo em uma cultura valiosa.

<Callout type="tip">
Considere remover uma mutação menos valiosa (como Úmida ou Congelada) de uma cultura de alta raridade se você tiver uma boa chance de adquirir uma mutação Arco-Íris ou Radioativa durante um evento.
</Callout>
