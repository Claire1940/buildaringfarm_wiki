<Section icon={Trophy} title="Guia Passo a Passo para Ajustar as Configurações">
Siga estes passos simples para configurar rapidamente as configurações do seu controle em Build A Ring Farm para um desempenho ideal no seu PS4 ou PS5.

<Steps>
  <Step num="1" title="Inicie Build A Ring Farm">
    Inicie a experiência Build A Ring Farm no Roblox a partir do seu console PS4 ou PS5. Certifique-se de que o jogo esteja totalmente carregado para acessar todos os recursos.
  </Step>
  <Step num="2" title="Abra o Menu de Configurações">
    Uma vez no jogo, pressione o **touchpad** no seu controle PlayStation. Isso abrirá um menu rápido. A partir daí, pressione o botão **Options** para abrir a interface principal de configurações.
  </Step>
  <Step num="3" title="Navegue para as Configurações de Exibição e Controle">
    No menu de configurações, procure o ícone 'Configurações', geralmente localizado no canto superior esquerdo. Clique nele para revelar uma lista de opções ajustáveis, incluindo gráficos, som e preferências de controle.
  </Step>
  <Step num="4" title="Configure Gráficos e Desempenho">
    - **Modo Gráfico**: Mude para 'Manual' e deslize a barra de qualidade para 'Baixa' se estiver experimentando quedas de taxa de quadros ou lag.
    - **Reduzir Movimento**: Ative esta opção 'Ligado' para suavizar ainda mais o desempenho visual, especialmente em sistemas mais antigos.
  </Step>
  <Step num="5" title="Ajuste as Configurações de Controle e Conforto">
    - **Vibração**: Localize a opção 'Vibração' e desligue-a para uma experiência de jogo mais focada.
    - **Sensibilidade**: Ajuste o controle deslizante 'Sensibilidade' para o seu nível preferido. Uma configuração em torno do meio é geralmente um bom ponto de partida para um controle equilibrado em Build A Ring Farm.
    - **Inverter Câmera**: Certifique-se de que 'Inverter Câmera' esteja 'Desligado', a menos que você prefira especificamente controles de câmera invertidos.
  </Step>
</Steps>
